3. API Endpoints
Create these endpoints:

GET /api/health
Returns backend status.

POST /api/companies
Creates a company with:
- company name
- optional description
- exactly 5 logo images

GET /api/companies
Returns all companies with their uploaded logo images.

GET /api/companies/:id
Returns one company with its images.

DELETE /api/companies/:id
Deletes company and its images from database and file storage.

POST /api/recognize/demo
Accepts one uploaded image and saves it as a recognition test.
Do not run AI yet.
Return a demo response:
- status: Demo Mode
- message: Upload received successfully. Real AI recognition will be added in Phase 3.
- uploaded image path
- similarity_score: null
- predicted_company: null
